PMP - Plan de Management de Projet

A la racine du dossier, vous pourrez consulter le management de projet dans son ensemble (GP-ProjetQCM.mpp) consultable avec Microsoft MS-Project. Vous pourrez y consulter plus facilement les éléments de planifications.

Composition:
	- Base de données : vous y trouverez toutes les informations techniques concernant la base de données
	- Documentation : vous y trouverez tous les cahiers de spécifications ainsi que le cahier des charges.
	- Maquette : vous y trouverez un exemple fonctionnel de l'application tel qu'elle sera conçue.
	- Planification: vous y trouverez tous les éléments utilisés pour la planification du projet
		-> WBS
		-> PERT
		-> GANTT
	- Test : Vous y trouverez tous les cas de test envisagés
	- User_case : Vous y trouverez tous les cas d'utilisations
