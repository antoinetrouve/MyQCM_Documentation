A la racine du dossier, vous pourrez consulter le management de projet dans son ensemble (GP-ProjetQCM.mpp) consultable avec Microsoft MS-Project. Vous pourrez y consulter plus facilement les éléments de planifications.

ARCHITECTURE
	\PMP
		\ARCHITECTURE
			\NETWORK
			\WEBSERVICE
		\BASE DE DONNEES
			\SERVER
			\TERMINAL
		\DOCUMENTATION
			\NAMING CONVENTION
			\SPECIFICATIONS

				\GENERAL SPECIFICATIONS
				\TECHNICALS SPECIFICATIONS

			\ETAT AVANCEMENT
			\PLANIFICATION
		\SCHEME UML

DETAILS
	\PMP
		\ARCHITECTURE
			\NETWORK
				+ Vous y trouverez le schema réseau de l'ensemble de la solution.
			\WEBSERVICE
				+ Vous y trouverez le schema du fonctionneement global du Webservice
		\BASE DE DONNEES
			\SERVER
				+ Vous y trouverez le diagramme de classe de la base de données située sur le serveur Web
			\TERMINAL
				+ Vous y trouverez le diagramme de classe de la base de données située sur le terminal mobile
		\DOCUMENTATION
			\NAMING CONVENTION
				+ Vous y trouverez les conventions de nommages utilisées pour le développement de l'ensemble de l solution
			\SPECIFICATIONS
				\FUNCTIONALS SPECIFICATIONS
					+ Vous y trouverez le cahier des spécifications générales
				\GENERAL SPECIFICATIONS
					+ Vous y trouverez le cahier des charges
				\TECHNICALS SPECIFICATIONS
					+ Vous y trouverez le cahier des spécifications détaillées
			\TEST SPECIFICATIONS
				+ Vou y trouverez le cahier de test fonctionnel pour la solution
		\JSON FLOW
			+ Vous y trouverez l'architecture des flux telle qu'elle est conçue pour chacun des flux Json
		\MAQUETTE
			\ADMINISTRATION
				+ Vous y trouverez la maquette non-fonctionnelle de l'interface d'administration
			\APPLICATION
				+ vous y trouverez un exemple fonctionnel de l'application tel qu'elle sera conçue.
		\PROJECT MANAGEMENT
			\ETAT AVANCEMENT
				+ Vous y trouverez les etats d'avancements par date d'analyse
			\PLANIFICATION
				+vous y trouverez tous les éléments utilisés pour la planification du projet
		\SCHEME UML
			+ Vous y trouverez tous les schemas UML par type d'UML
		\TOOLS
			+ Vous y trouverez les outils libres permettant de consulter notamment les schemas UML.
